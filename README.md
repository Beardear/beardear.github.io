Personal portfolio website for Moran Xu — Applied Research Scientist and Senior Algorithm Engineer at Carl Zeiss, Inc.

Built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## About

This site showcases research, projects, and publications in Computer Vision, Computational Imaging, Agentic AI, and Multi-modal Understanding. It features a fixed sidebar layout with light/dark/system theme support.

## Structure

- `index.html` — Main page with About, Projects, Publications, and Resume sections
- `_layouts/` — Page layouts (`home`, `project`, `default`)
- `_includes/` — Shared partials (`header`, `footer`, `author-sidebar`)
- `projects/` — Individual project pages
- `assets/styles.scss` — All styles with CSS custom properties for theming
- `assets/images/` — Profile photo, project covers, publication thumbnails

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open [http://localhost:4000](http://localhost:4000).
